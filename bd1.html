CUESTIONARIO


  1. ¿Qué es entonces una base de datos - DB?

    Una base de datos es una colección de archivos interrelacionados. Dichos archivos están almacenados y relacionados de manera eficiente. Para su manejo se utiliza un sistema administrador de bases de datos.

  2. ¿En qué casos es conveniente usar bases de datos?

    Las bases de datos son convenientes cuando se requiere almacenar grandes cantidades de datos y que a la vez están relacionados de alguna manera. Igualmente, dichos datos pretenden ser accedidos por muchas entidades; ya sea, a la vez o a través de un periodo de tiempo.

  3. ¿Qué es un sistema de gestión de base de datos - DBMS?

    Es un programa que proporciona la interfaz para falicitar el acceso a la información guardada en una base de datos; ya sea para consultas o para escrituras. Dicho programa lo hace de manera eficiente y evitando problemas de concurrencia por medio de algoritmos especializados complejos para este propósito.

  4. Enlista y explica las funciones/responsabilidades que tienen los DBMS.

    Las principales responsabilidades de los DBMS son:
    - Simplificar y facilitar el acceso a los datos.
    - Evitar problemas de concurrencia.
    - Traducir las sentencias DML a comandos para el sistema de archivos de bajo nivel.
    - Almacenamiento, recuperación y actualización de la base de datos.

  5. Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

    Las bases de datos son de gran utilidad y pueden aplicarse a muchas cosas en empresas y servicios de la vida cotidiana; sin embargo, no siempre tiene que ser para una empresa o servicio. Siempre que se tengan las características para que exista una BD se puede aplicar. Un ejemplo de un proyecto donde me hubiera sido útil una base de datos es en una carrera que organicé en preparatoria. En dicha carrera habían muchos participantes, todos registrados y con un boleto, todos con diferentes pesos, edades, estaturas etc.
    Otro ejemplo sería nuestro proyecto de FDI.
