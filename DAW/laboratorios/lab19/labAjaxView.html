¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications? - Es sumamente importante; ya que permite el uso de solicitudes asíncronas y como consecuencia, una mejor experiencia de usuario, más enriquecida. ¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor? - La lógica se procesa de manera local por javascript; sin embargo, las validaciones se realizan del lado del servidor por medio de php. ¿Qué es JSON? - JavaScript Object Notation, se una manera legible de pasar objetos en forma de texto desde JS al servidor esto permite que dicha acción se realice de manera ligera y se puede pasar de JSON a objeto de JS y viceversa.
